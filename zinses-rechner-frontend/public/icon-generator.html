📋 使用说明

  1. 点击上面的"生成图标"链接
  2. 在新页面中右键点击图标，选择"另存为图片"
  3. 保存为对应的文件名（如 pwa-192x192.png）
  4. 将生成的PNG文件放在 public/ 目录下
  5. 重新构建应用: npm run build
  6. 重新部署: npm run deploy
